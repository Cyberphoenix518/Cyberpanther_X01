# Cyberpanther_X01
CyberPanther_X01 is a comprehensive, all-in-one penetration testing framework  designed for authorized cybersecurity professionals. Combines stealth evasion,  DoS testing, bruteforce capabilities, web exploitation, and automated reporting  in a single, production-ready Bash script with intuitive TUI interface.
How to run this tool
1. git clone https://github.com/Cyberphoenix518/Cyberpanther_X01.git
2. cd Cyberpanther_X01
3. bash Cyberpanther.sh
